CSS 指层叠样式表,通过仅仅编辑一个简单的 CSS 文档，使你有能力同时改变站点中所有页面的布局和外观。

语法:	
	css变量名 html元素{html元素属性:值;...}	eg:#sidebar h1{color:red; font-size:14px;}
使用:
	每个标签中设置class(类选择器)或id(ID选择器),属性选择器
	不同:
		id 选择器以 "#" 来定义。	eg:#red {color:red;}  使用:<p id="red">
		类选择器以"."来定义。		eg:.red {color:red;}  使用:<p class="red">
		属性选择器对带有指定属性的 HTML 元素设置样式。	eg:[title]{color:red;}
